Functionality

SYSTEM musi umożliwiać integrację logowania do SYSTEMU z bazą LDAP (S.M.A.R.T)

Online help	Will online help be required?
Wykonawca udostępni internetowy system zgłoszeń czynny 24 godziny na dobę; (S.M.A.R.T.)

Printing	Is printing capability required?
SYSTEM musi  umożliwiać  podgląd,  wydruk  i  zapis  w  wersji  elektronicznej  wszystkich raportów zdefiniowanych przez dostawcę (S.M.A.R?.T?)

Reporting	Is reporting capability required? 
SYSTEM musi zapewniać użytkownikowi wszystkie potrzebne zestawienia: (S? M A R? T?)
Standardowe raporty dostępne bezpośrednio po wdrożeniu 
Niestandardowe, definiowane przez użytkownika 
Sprawozdania  sporządzane  w  trybie  obowiązującym dla  urzędów  państwowych  i uczelni, zgodnie z ich instrukcjami i przepisami

Response time	Are there any constraints on the time the system takes to respond to particular events, such as user interaction?
SYSTEM musi zachować taką samą stabilność i wydajność pracy niezależnie od wzrostu liczby użytkowników
(S? M A? R T?)

Throughput	Are there any special requirements regarding throughput of data supported by the system?
SYSTEM
musi być dostarczony z platformą integracyjną - „szyną danych” umożliwiającą zarządzanie komunikacją z aplikacjami zewnętrznymi i wykorzystującą standardy wymiany informacji: EDI, WebServices lub równoważne.
(SMAR T?)



Supportability

Adaptability	Are there any special requirements regarding adaptation of the software (including upgrading)?
SYSTEM musi  posiadać  specyfikację  interfejsu  API  umożliwiającego  rozbudowę SYSTEMU o kolejne komponenty przez różnych dostawców, w tym przez Zamawiającego
(SMAR)

Localization 	Which human languages need to be supported?
SYSTEM musi  posiadać  polskie  interfejsy  użytkownika,  w  przypadku  interfejsów administracyjnych  i  do  komponentów  środowiska  dopuszczalne  są  interfejsy  w  języku angielskim
(SMAR)

Maintainability	Are there any special requirements regarding system maintenance?
Serwery  aplikacji  muszą  pracować poprawnie  na  platformach  Win64  lub  Linux  (bądź pochodnych), jednakże  zarządzanie  nimi  musi  być  wykonywane  ze  stacji  roboczych pracujących na platformach Win32 lub Win64.
(SMAR)
